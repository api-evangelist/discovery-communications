# Discovery Communications

This repository tracks APIs, SDKs, and other developer resources for Discovery Communications, now part of Warner Bros. Discovery (WBD) following the 2022 merger with WarnerMedia.

Warner Bros. Discovery is a leading global media and entertainment company whose portfolio spans HBO Max, Discovery Channel, CNN, Warner Bros. film studios, DC, Food Network, HGTV, and other television, film, and streaming brands.

## APIs

No public developer APIs have been documented at the corporate level. Technical and partnership inquiries are handled at the individual brand or business-unit level.
